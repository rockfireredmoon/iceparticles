Particle Scripts

Previewing Scripts

When editing a script file, all the scripts contained within it will be listed in top half of the script editor window.

Each script has a checkbox next to it. Click to this to turn previewing of that script on an off.

Note, it is currently only possible to edit one script file at a time. When switch to another script file, previews of all the current scripts will be turned off, and any unsaved changes will be discarded. Do not worry, you will be asked to confirm the action if this happens!

Creating A New Script

Click on the New Script button at the very top of the editor window. You will be asked to provide a name for the script.

Note, Planet Forever script names usually being with Par-Something. Please follow this convention.

Deleting A Script

First select the script you wish to delete by clicking it. The row will be highlighed in blue. Then at the top of the Particle Editor Window is a Delete button. Click on this and you will be asked to confirm the deletion. Click on Ok and the script will be deleted, and any current previews of that script will be stopped.

Alternatively, you can right click an individual script to show the script context menu, and click Delete.

Copy And Paste

You can copy each individual script to the system clipboard by right clicking it and chosing the Copy menu item. You may then paste this script text into any other external editor and make changes to the script directly.

When done, copy the changed script back into the clipboard, then use the Paste menu item to put the changes back in to the script file you are editing. The UI will be updated accordingly.
